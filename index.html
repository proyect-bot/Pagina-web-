Guía Completa para Crear una Página Web con HTML y CSS

1. ¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado utilizado para estructurar contenido en la web. Define elementos como encabezados, párrafos, imágenes y enlaces.

2. Estructura Básica de HTML

Todo documento HTML debe comenzar con la siguiente estructura:

<!DOCTYPE html> 
<html lang="es"> 
<head> 
    <meta charset="UTF-8"> 
    <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
    <title>Mi Página Web</title> 
</head>
<body> 
    <h1>¡Hola Mundo!</h1>
</body>
</html>

3. ¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje utilizado para darle estilo a los documentos HTML. Permite cambiar colores, fuentes, márgenes y más.

4. Cómo Agregar Estilos con CSS

Podemos darle estilo a nuestra página con CSS de diferentes maneras:

  • CSS en línea: Se aplica directamente en la etiqueta HTML con el atributo style.
  • CSS interno: Se coloca dentro de la etiqueta <style> en el <head>.
  • CSS externo: Se usa un archivo separado con extensión .css y se enlaza con <link>.
/* Ejemplo de CSS interno */
body {
    background-color: #f4f4f4;
    font-family: Arial, sans-serif;
    color: #333;
}

5. Creando una Página Completa

Ahora combinamos HTML y CSS para hacer una página sencilla con un encabezado y un contenedor de contenido.

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi Página Completa</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            background-color: #f4f4f4;
            color: #333;
            text-align: center;
        }
        header {
            background: #0073e6;
            color: white;
            padding: 20px;
        }
    </style>
</head>
<body>
    <header>
        <h1>Bienvenido a Mi Página</h1>
    </header>
    <p>¡Esto es una página web básica con HTML y CSS!</p>
</body>
</html>

6. Conclusión

Con HTML y CSS puedes crear páginas web personalizadas y bien diseñadas. ¡Practica y experimenta con diferentes estilos para mejorar tus habilidades!
